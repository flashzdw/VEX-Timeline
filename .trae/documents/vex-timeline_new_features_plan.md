# VEX-Timeline 新功能计划

## 功能需求
1. 添加图片功能 + 选择具体时间
2. 月历模式下用卡片展示记录
3. 重要性选项 + 筛选功能

## 实现步骤

### 1. 数据库升级
- 更新 IndexedDB 结构支持新字段
- 迁移现有数据

### 2. 表单增强
- 添加图片上传功能
- 添加时间选择器
- 添加重要性选择（高/中/低）

### 3. 时间轴优化
- 显示重要性颜色标记
- 支持图片预览
- 添加筛选工具栏

### 4. 月历视图重构
- 点击日期显示记录卡片
- 展示完整记录信息

### 5. 样式更新
- 定义重要性颜色
- 卡片样式
- 筛选栏样式
